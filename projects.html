My Projects

Here are some of the projects I’ve been working on recently:

Computational Neuroscience

Computational neuroscience project exploring the human brain's robustness to visual noise. I was part of a pod at Neuromatch Academy (pod Lively anat). Check it out here: GitHub.

Eurogene Project

An exploration of genomes from diverse ancient and modern populations. Check it out on GitHub.

PINN library

A small, easy-to-use Python library for solving PDEs with Physics-Informed Neural Networks (PINNs). Check it out on GitHub.

BCI, Motor Imagery

This project uses data from the BCI Competition IV, Dataset 1, which focuses on motor imagery EEG signals. The dataset was recorded from healthy subjects performing motor imagery tasks (e.g., imagining left hand, right hand, or foot movement) without feedback. Check it out on GitHub.
